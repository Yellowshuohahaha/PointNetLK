# PointNetLK: Point Cloud Registration using PointNet

Source Code Author: Vinit Sarode

### Code use:
1. Train PointNet:
> python train_classifier.py
2. Train PointNetLK:
> python main.py
3. Test PointNetLK;
> python test.py

### Requirements:
1. PyTorch
2. ModelNet40 [Link](https://modelnet.cs.princeton.edu/)
3. SciPy
